# OpenFengHuo（开放烽火）

> 对「烽火互联」真人CS对战App（包名 `com.fh.mobile`）进行协议逆向，成功在本地复刻登录模块，支持连接自建私服。

---

## 项目概述

**目标**：在不依赖原厂服务器的情况下，复刻烽火互联CS对战系统的核心通信协议，搭建私有服务器。

**已完成**：

- 登录接口  完整逆向
- JWT Token 伪造（HS512）
- Python FastAPI 私服登录模块（开箱即用）
- 武器列表 / 战绩 / 血量接口已占位待填充

**进行中**：

- 武器系统、血量逻辑的详细逆向
- Frida Host 重定向脚本（App流量劫持）
- BLE蓝牙硬件协议分析（枪-手机通信）
